HTML

HTML is short for Hypertext Markup Language. HTML describes structure and content. It is the standard markup language for creating Web pages and it consists of a series of elements. HTML elements tell the browser how to display the content. HTML elements label pieces of content for examples, "this is a heading", "this is a paragraph", "this is a link", etc.
